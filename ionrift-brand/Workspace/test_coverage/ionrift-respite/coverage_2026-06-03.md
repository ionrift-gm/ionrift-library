## Why These Tests Matter

This batch targets six modules that sit at the foundation of the Ionrift suite. **PackManifestSchema** validates every content pack before import — a bug here silently corrupts pack installations. **CompendiumConfigGuard** prevents a known Foundry crash (#13225) during world launch — the pure `planRepair()` function now has complete coverage for every repair scenario. **TerrainRegistry** is the canonical terrain source for every downstream module; category normalization bugs silently miscategorize encounters. **SettingsVisibility** guards against GM settings leaking to players — a one-line regression in the `scope` check would expose world-level configuration. **LegacyAssetSweeper** utility methods underpin the orphan file detection used by the Patreon Library cleanup panel; the browse-target guard specifically prevents false positives from Foundry v13's silent parent-fallback behavior. The **ItemMintingService** extensions close gaps around `guardPatch` (Quartermaster promotion path), the `-=` deletion DSL (fixed in v2.1.5), and formula/attunement/damage validation that were absent from the initial test file.
